Convenient Discussions - Development Server

Note: This page is for development only. The actual app runs on MediaWiki pages.

Development with HMR

To load Convenient Discussions with Hot Module Replacement on a MediaWiki page:

  1. Make sure this dev server is running (npm start)
  2. Navigate to a MediaWiki talk page (e.g., test.wikipedia.org/wiki/User_talk:JWBTH)
  3. Open the browser console (F12)
  4. Paste and run this code:
// Load Convenient Discussions with HMR
const script = document.createElement('script');
script.type = 'module';
script.src = 'http://localhost:9000/src/app.js';
document.head.appendChild(script);

The app will load and you'll have full HMR support:

  • Edit source files in your editor
  • Changes are reflected instantly without page reload
  • Application state is preserved during updates

Alternative: One-liner

For quick testing, use this one-liner in the console:

document.head.appendChild(Object.assign(document.createElement('script'), {type: 'module', src: 'http://localhost:9000/src/app.js'}));

Production Build

For production deployment, use:

npm run build

This creates the IIFE bundle at dist/convenientDiscussions.js that can be loaded with mw.loader.load().
